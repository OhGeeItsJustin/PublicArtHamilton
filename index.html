Beta Project for HTML5 class. This application is used for users that want to find the nearest public art around Hamilton. This application creates a map on the screen. For any screen size that aren't mobile has filter buttons above the map that create markers on the map. Application gets users location and displays it on the map with a marker and displays the users latitude and longitude below the map. Below the latitude and longitude are two extra buttons that display the description of the program and information about the creator. At the end of the application their is a footer that displays who maintains the page and the contact information. When on a mobile device the screen only displays the header, map, three filter buttons below the map and a footer.
